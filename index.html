Na sessão de projetos você encontrará outros projetos que eu realizei e estou a realizar

Projeto Mobile - Sistema de Autenticação QR-ID para Extensões das Etecs desenvolvido em React Native O projeto consiste no desenvolvimento de um aplicativo mobile destinado à autenticação de alunos e professores nas extensões das Etecs do Centro Paula Souza em Campinas. Atualmente, essas extensões não contam com um sistema eficaz de controle de acesso, o que gera uma preocupação com a segurança. Para solucionar esse problema, esta sendo desenvolvido um aplicativo, denominado SIE. O SIE tem como objetivo criar um ID único para cada aluno e professor por meio de códigos QR. Além disso, o aplicativo fornecerá ao pessoal de segurança um scanner restrito para realizar a autenticação do QR-ID. O aplicativo possui quatro telas principais: tela de login, tela de scanner, tela de exibição do QR-ID e tela do administrador. Com essa abordagem multidisciplinar e uma estrutura clara de trabalho, o objetivo é fornecer uma solução eficaz para melhorar a segurança nas extensões das Etecs em Campinas.
